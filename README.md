# Introduction
Amazon Simple Storage Service (S3), a service offered by Amazon Web Services, provides a highly scalable and cost-effective solution for hosting static websites. Static websites consist of fixed content such as HTML, CSS, JavaScript, and media files, without the need for server-side processing. # Objective
The main objective of this project is to:
•	Create an S3 bucket configured for static website hosting 
•	Upload website files (HTML, CSS, etc.) from a GitHub repository 
•	Enable public access to allow users to view the website 
•	Successfully host and access the website via a public endpoint
